# 🚀 Harshit Yadav — Personal Portfolio

A clean, minimal, and responsive personal portfolio website built with pure **HTML, CSS, and JavaScript** — no frameworks, no dependencies.

---

## ✨ Features

- 🌗 **Dark / Light mode** toggle
- 🎨 **Orange accent theme** with ambient background glow
- 📱 **Fully responsive** — works on mobile, tablet, and desktop
- ✨ **Scroll reveal animations** — sections fade in as you scroll
- 🔤 **Custom typography** — Syne (headings) + DM Sans (body)
- ⚡ **Zero dependencies** — single HTML file, no build tools needed

---

## 🗂️ Sections

| Section | Description |
|---|---|
| **Hero** | Introduction, tagline, and CTA buttons |
| **About** | Background, university, and builder mindset |
| **Skills** | Tech stack with icon grid |
| **Projects** | Featured projects with live/code links |
| **Contact** | Email, GitHub, LinkedIn, and X (Twitter) |

---

## 🛠️ Tech Stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, flexbox, animations
- **Vanilla JavaScript** — theme toggle, Intersection Observer for scroll reveal
- **Google Fonts** — Syne & DM Sans

## 🚀 Getting Started

No build step needed. Just clone and open:

```bash
git clone https://github.com/harshityadav8610-eng/portfolio
cd portfolio
# Open index.html in your browser
```

Or deploy instantly to **Vercel / Netlify / GitHub Pages** by uploading the folder.
